1. AND
    입력 신호가 모두 1일때만 1출력
    Y = A*B /  Y = AB
2. OR
    입력 신호 중 1개만 1이어도 1출력
    Y = A+B
3. BUFFER
    입력 신호를 그대로 출력
    Y = A
4. NOT
    입력 신호를 반대로 변환하여 출력
    Y = !A
5. NAND
    NOT+AND(AND의 부정)
6. NOR
    NOT+OR(OR의 부정)
7. XOR
    입력 신호가 같으면 0 , 다르면 1 출력
8. XNOR
    NOT+XOR
    입력 신호가 같으면 1, 다르면 0 출력
